진입 게이트

매수 신호가 실제 주문이 되기까지 — 5단계 관문

하나라도 통과 실패하면 주문이 발행되지 않습니다.

📡 신호 발생 (스크리너 or 배치 스캔)
↓
G1
마켓 레짐 필터
• 약세장 (KOSDAQ -2%+) → 모든 신규 진입 차단
• ranging 레짐 → RSI2 / SEPA만 허용
• 아침 LLM 진단 → 실시간 체제 조정
G2
크로스 검증 (9규칙)
• 기관+외국인 동시 순매도 → 즉시 차단
• 동일 섹터 3종목+ → 즉시 차단
• 추격(ATR 1.5x) / RSI>70 → 감점
• 적자+고PBR / MA200 하방 → 감점
• 감점 후 50점 미만 → 차단
G3
리스크 매니저
• 일일 손실 -5% 초과 → 당일 전면 중단
• 당일 청산 종목 30분 쿨다운
• 포트폴리오 동기화 3회 실패 → 차단
G4
LLM 이중 검증
• 발동 조건: 점수 85+ AND 비강세장
• LLM이 뉴스·수급·레짐 맥락 종합 판단
• 하루 최대 5회 (비용 제한)
• LLM 거부 → 진입 차단
G5
포지션 사이징
• 단일 종목 최대 28% 한도
• 코어홀딩 예산 30% 별도 보호
• 가용 현금 부족 시 자동 스킵

CORE PRINCIPLES — 21개

모든 매매에 적용되는 수정 불가 규칙입니다. 예외는 없습니다.

4개
CORE-001 HIGH
손절은 신성하다 — 예외 없이 즉시 청산
작은 손실 10번이 큰 손실 1번보다 낫다. ATR×2 동적 손절 (3.5~6%).
CORE-002 HIGH
일일 손실 -5% 초과 시 당일 매수 전면 중단
틸트 상태에서의 복구 매매는 손실을 키운다.
CORE-003 HIGH
단일 종목 비중 28% 초과 금지
분산이 생존이다. 한 종목 실패가 포트폴리오를 무너뜨리면 안 된다.
CORE-013 HIGH
포트폴리오 동기화 3회 연속 실패 시 매수 전면 차단
잔고를 모르는 상태에서 매수하면 과잉 노출된다.

CORE-004 HIGH

CORE-005 HIGH
기관+외국인 동시 순매도 테마주 진입 금지
개인만 매수하는 테마는 단기 급등 후 급락한다.
CORE-006 MED
장초반(09:05~10:00) +4~5% 초과 급등 → 진입 보류
장초반 과열은 10시 이후 눌림이 온다.
CORE-007 HIGH
약세장에서는 SEPA/RSI2만 허용 — 테마/갭 즉시 차단
약세장에서 공격적 전략은 손절 확률이 2배.
CORE-014 MED
RSI(14) > 70 + SEPA 추세 진입 → -10점 감점
기술적 과매수에서 추세 진입은 고점 물림 위험.
CORE-015 MED
MA200 하방 종목 추세 진입 → -10점 감점
장기 하락 추세 종목의 단기 반등은 지속력이 약하다.
CORE-016 HIGH
MA200 대비 +80% 과확장 종목 SEPA 진입 차단
60일 급등 후 추격은 고점 물림의 전형.
CORE-017 MED
시총 상위 20 대형주 → 테마 전략 제외
대형주는 테마 모멘텀이 약해 +2~3%가 한계.

3가지 설계 철학
🚪
Exit First
청산 조건은 매수 신호보다 항상 우선 처리된다. 수익을 지키는 것이 새 수익을 버는 것보다 중요하다.
🔄
Context-Aware
시장 레짐에 따라 손절폭, 익절 목표, 허용 전략이 자동 조정된다. 고정 파라미터로 모든 시장에 대응하지 않는다.
🧬
Learning Loop
매 거래가 기록되고 LLM이 패턴을 추출하며 다음 진입 점수 보정에 반영된다. 같은 실수를 반복하지 않는 구조.
